# Pokedex Full Stack Application

A full-stack Pokédex web application built using modern technologies like Next.js, tRPC, Prisma, and PostgreSQL. This app allows users to search Pokémon, view details, and filter them based on type.

---

## Features

- Search **single Pokémon** by name
- View **all Pokémon**
- Filter Pokémon by **type**
- Display Pokémon **sprite images**
- Fast API using **tRPC**
- Database integration with **Prisma + PostgreSQL**

---

##  Tech Stack

- Frontend: Next.js, TypeScript, Material UI
- Backend: tRPC
- ORM: Prisma
- Database: PostgreSQL
